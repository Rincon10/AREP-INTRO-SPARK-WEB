API

Este es un API permite calcular valores de temperatura equivalentes a través de varias escalas de temperatura comúnmente utilizadas.Esta API posee unicamente 2 endpoints , donde cada uno de ellos sirve para un propósito diferente cubriendo los diferentes casos de uso caso de uso.

Los endpoints incluyen funcionalidades como recibir valores de las escalas Celsius y Fahrenheit, transformandolas respectivamente segun se necesite.

URL Base

https://calcapp-backend.herokuapp.com/api/v1

Respuesta API

Los datos se devuelven en un formato JSON estándar y se pueden volver a analizar fácilmente en cualquier lenguaje de programación.

Ejemplo de respuesta: a continuación se mostrara un ejemplo de una respuesta de API que brinda información sobre la conversion realizada en una de las diferentes escalas, en esta respuesta encontraremos él código HTTP el cual nos dirá el estado de la petición, el valor transformado y por último la unidad a la cual se transformó. 

    {
        "status_code":200, 
        "value": 1, 
        "unit":"fahrenheit"
    }

Endpoints disponibles

La API viene con una serie de endpoints donde cada uno de ellos proporciona una funcionalidad diferente.

  • endpoint para conversión fahrenheit a celcius: este endpoint devuelve la conversión en tiempo real de nuestros grados celsius a fahrenheit, para esto debemos de usar el siguiente path.

    Solicitud API:

    https://calcapp-backend.herokuapp.com/api/v1/celsius/{fahrenheit}

    Parámetros de solicitud:

    Parametro Descripción
    fahrenheit [obligatorio] Grados fahrenheit que se convertiran en celsius.

  • endpoint para conversión celcius a fahrenheit: este endpoint devuelve la conversión en tiempo real de nuestros grados fahrenheit a celsius, para esto debemos de usar el siguiente path.

    Solicitud API:

    https://calcapp-backend.herokuapp.com/api/v1/fahrenheit/{celsius}

    Parámetros de solicitud:

    Parametro Descripción
    celsius [obligatorio] Grados celsius que se convertiran en fahrenheit.

Control de acceso / CORS

El uso compartido de recursos de origen cruzado (CORS) y los encabezados de control de acceso también son compatibles con la API de conversor de unidades. Esto le brinda la posibilidad de utilizar la API de tipos de cambio a través de solicitudes HTTP de origen cruzado

Errores potenciales

Cuando enviamos solicitudes de API y esta llamada falla por algún motivo, se devuelve un error JSON.

Error de ejemplo:

{
    "status_code":400, 
    "value": -1, 
    "unit":"none"
}
